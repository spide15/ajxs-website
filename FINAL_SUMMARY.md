## 🎨 CUSTOMIZATION GUIDE

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: '#dc2626',    // Your primary color
  secondary: '#ea580c',  // Your secondary color
}
```

### Change Logo
Replace in `ajxs-website.jsx`:
```javascript
<div className="w-10 h-10 bg-gradient-to-br from-red-600 to-orange-500">
  {/* Replace with <img src="logo.png" /> */}
</div>
```

### Add More Products
Edit `PRODUCTS` array in `ajxs-website.jsx`:
```javascript
const PRODUCTS = [
  { "id": "49", "name": "Your New Product", "category": "GIFT" },
  // ... add more
];
```

### Update Spotlight Offers
Edit `SPOTLIGHT_OFFERS` array:
```javascript
const SPOTLIGHT_OFFERS = [
  {
    id: 5,
    title: "Your New Offer",
    description: "Description here",
    originalPrice: "₹999",
    discountedPrice: "₹599",
    discount: "40%",
    endDate: "2025-12-31",
    image: "https://your-image-url.jpg"
  }
];
```

## 🆘 TROUBLESHOOTING QUICK FIXES

| Problem | Solution |
|---------|----------|
| npm install fails | `rm -rf node_modules && npm install` |
| Port 5173 in use | `npm run dev -- --port 5174` |
| Images not loading | Check internet, clear cache |
| Build fails | Check for syntax errors with `npm run build` |
| Domain not working | Wait 24-48 hours for DNS, verify DNS records |
| WhatsApp not working | Check phone number format |
